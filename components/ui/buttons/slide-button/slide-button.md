# SlideButton

A slide-to-confirm button with drag gesture. Drag the handle right to submit.

![Preview](preview.png)

## Installation

```bash
npx shadcn@latest add https://raw.githubusercontent.com/o1hive/design-vault/main/registry/slide-button.json
```

## Usage

```tsx
import { SlideButton } from "@/components/ui/buttons/slide-button"

export function Example() {
  return <SlideButton />
}
```

## Props

SlideButton accepts all shadcn `ButtonProps`.
